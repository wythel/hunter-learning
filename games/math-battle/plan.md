# Math Battle — 算數闖關遊戲 計畫

## 概述

寶可夢風格的回合制戰鬥遊戲。玩家扮演勇者，透過答對加減法題目對怪物發動攻擊；答錯則被怪物反擊扣血。全部答完後結算勝負。

目標年齡：5歲以上
學習目標：個位數 / 雙位數加減法

---

## 遊戲流程

```
開始畫面
  └─ 選難度（簡單 / 困難）
  └─ 選題數（5 / 10 / 15 題）
      └─ 進入戰鬥畫面
            └─ 顯示題目
            └─ 玩家輸入答案
                  ├─ 答對 → 玩家攻擊怪物特效 → 怪物扣血
                  └─ 答錯 → 怪物攻擊玩家特效 → 玩家扣血
            └─ 繼續下一題（或怪物血量歸零 → 換下一隻怪）
      └─ 所有題目結束 → 結算畫面（得分、答對率）
```

---

## 難度設定

| 難度 | 數字範圍 | 運算 |
|------|----------|------|
| 簡單 | 1 ~ 9    | 加法、減法（結果 >= 0）|
| 困難 | 10 ~ 99  | 加法、減法（結果 >= 0）|

## 特效規劃

| 事件 | 特效 |
|------|------|
| 答對（玩家攻擊）| 玩家角色向前衝，怪物畫面閃爍紅色 + 抖動，傷害數字飄出 |
| 答錯（被攻擊）  | 怪物向前衝，玩家畫面閃爍紅色 + 抖動，扣血數字飄出 |
| 怪物死亡        | 怪物漸漸消失（fade out）+ 勝利音效 |
| 玩家HP歸零      | 玩家角色淡出，進入遊戲結束（可重來）|

特效實作方式：CSS animation + keyframes（不需要 canvas）

---

## 血量設計

- 玩家初始 HP：固定 3 顆愛心（❤️❤️❤️），答錯扣 1 顆，歸零則遊戲結束
- 怪物 HP：固定 3 格（答對 1 題打掉 1 格，打完換下一隻怪）
- 怪物種類：準備 3~5 種不同外觀（用 emoji 或簡單 CSS 繪製）

---

## 技術實作

- 純 HTML + CSS + JavaScript（無框架）
- 所有動畫用 CSS keyframes
- 角色圖示：先用 emoji（例如 🧙 勇者、🐲🐸👾 怪物），之後可替換成圖片
- 響應式：以 375px 寬度（iPhone SE）為基準設計，往上 scale
